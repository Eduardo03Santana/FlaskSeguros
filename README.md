# FlaskSeguros
Uma pagina web desenvolvida com Flask, para obtenção de nota no curso de extensão "Python Avançado Para Web Com Flask", ministrado pelo professor Me. Henrique Poyatos

O intuito desse projeto é desenvolver um CRUD básico com a ferramenta FLASK.

Descrição:

Admin:
O admin pode gerenciar os produtos, e seguradoras, podendo cadastrar, alterar e excluir um produtos, ou seguradora.


Admins são cadastrados diretamente no banco.

Usuario:
Usuarios podem contratar os seguros disponiveis para quem ele quiser.
Podendo posteriormente, alterar e deletar um seguro, ou um dos segurados sob sua gestão.

Usuarios podem se cadrastar através do site.
